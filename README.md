# EmbarcaTech
Este projeto visa a implementação de um sistema de entregas via drones, utilizando do Direct Memory Access (DMA) para a tomada de decisão em tempo real, com a finalidade de otimizar os ganhos e viabilizar o modelo de negócio proposto. Trata-se de um sistema que coordena múltiplos agentes: drones, aeroportos, clientes e lojas.
